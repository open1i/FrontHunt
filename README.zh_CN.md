# FrontHunt

[🇺🇸 English](./README.md) | [🇨🇳 中文](./README.zh-CN.md)

FrontHunt 是一个开源项目，提供最新的前端面试题和模拟面试工具。  
帮助前端开发者高效备战技术面试，包含按主题分类的面试题、详细解析及练习功能。

## 功能特色

- 实际且持续更新的前端面试题
- 按主题和难度分类
- 支持导出题目和答案为 PDF
- 模拟面试环境，带计时和评分功能
- 针对 PC 端优化的用户界面

## 快速开始

1. 克隆仓库：
   ```bash
   git clone https://github.com/open1i/FrontHunt.git
   ```
